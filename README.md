# gestion-de-stock-pharma
Le système de gestion de stock d’une pharmacie permet de suivre et contrôler les médicaments disponibles afin d’assurer un approvisionnement optimal. Il enregistre les entrées (achats) et les sorties (ventes), met à jour automatiquement les quantités et alerte en cas de rupture ou de produits proches de la date d’expiration.
